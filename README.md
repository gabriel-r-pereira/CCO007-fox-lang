# CCO007-fox-lang
Compilador desenvolvido com propósitos educacionais.

Dependências:
- Make
- Bison
- Flex
- GCC

Compilando um arquivo da linguagem Fox:
- No terminal, execute o comando
    "make -B"
- Execute o comando
    "./fox codigo.fox"
- Compile o código C gerado
    "gcc -o exec out.c"
- Execute o arquivo exec
    "./exec"
